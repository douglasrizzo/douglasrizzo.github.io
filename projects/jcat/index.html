jCAT

Java Web application for computerized adaptive testing application in tablets

A Java EE web application whose purpose is to apply both an electronic version and a computerized adaptive test of Provinha Brasil, a nation-wide educational evaluation for Brazilian students in the second year of basic school.

This was part of a R&D project financed by Unesco, which resulted in several publications along the following years (Alavarse et al., 2017; Alavarse et al., 2018; Travitzki et al., 2021).

    Abs

    Este artigo descreve um Teste Adaptativo Informatizado (TAI) para a Provinha Brasil – Leitura, com base na Teoria da Resposta ao Item. Detalha-se o funcionamento e processo de desenvolvimento do algoritmo. O TAI foi aplicado com o uso de tablets a 1.983 alunos dos 1o e 2o anos do ensino fundamental, em 15 escolas da Rede Municipal de Ensino de São Paulo. Os resultados confirmam a qualidade dos itens da Provinha Brasil, do trabalho realizado nas escolas e, sobretudo, do TAI. Em relaç ão à gestão do tempo de prova, conclui-se que há uma associação de mesmo sentido entre proficiência e tempo , mas só até certo ponto; os alunos tendem a demorar mais nos itens mais difíceis; essa tendência é mais intensa nos alunos mais proficientes, confirmando a hipótese de que eles tendem a gerir melhor o tempo de prova.
